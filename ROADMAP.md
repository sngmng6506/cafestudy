# Roadmap

아래 항목들은 앞으로 추가되면 좋은 기능 후보입니다. 관심 있는 사람이 원하는 항목을 골라 구현해도 됩니다.

## 구현 후보

### 카페 정보
- 모임이 열린 적 있는 카페 목록을 보여준다.
- 카페 이름, 위치, 방문 이력, 관련 모임을 함께 확인할 수 있게 한다.
- 네이버 장소 검색 결과와 연결해 카페 식별을 더 안정적으로 만든다.

### 카페 이력과 코멘트
- 완료된 모임의 카페를 카페 이력에 남긴다.
- 해당 카페에 참석한 적 있는 사람만 한줄 코멘트를 남길 수 있게 한다.
- 같은 카페가 다른 문자열로 쪼개지지 않도록 장소 ID 또는 정규화된 키를 도입한다.

### 모임 이력
- 시간이 지나 완료된 모임을 모임 이력 탭으로 이동시킨다.
- 완료 시점의 참석자 목록을 보존한다.
- 나중에 인증 사진이 올라오면 해당 모임 이력에 연결해 보여준다.

### 지도 연동
- 앱의 지도보기 버튼에 해당하는 장소 정보를 웹에서도 확인할 수 있게 한다.
- 웹 크롤링만으로 장소 좌표를 얻을 수 있는지 추가 확인한다.
- 불가능하면 장소명 기반 검색 API로 보완한다.

### 앱 자동화 API
- 웹에 없는 운영진 기능을 앱 자동화로 조작할 수 있는지 검토한다.
- 후보 기능: 모임 개설, 모임 수정, 참석자 확인.
- 에뮬레이터, 실기기, 미니 PC 중 어떤 운영 방식이 현실적인지 결정한다.
